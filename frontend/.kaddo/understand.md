## Copy/Paste Instructions

Start with **capability-agent**:

1. Open your preferred LLM chat (Claude, ChatGPT, Cursor, Copilot, Windsurf…).
2. Paste the context pack: `.kaddo/context-pack.md`
3. Paste the agent prompt: `knowledge/agents/product/capability-agent.md`
4. Ask the LLM to produce: `knowledge/product/capabilities.md`
5. Save the result in: `knowledge/product/capabilities.md`

## Next Steps

After saving `knowledge/product/capabilities.md`, continue with **architecture-agent** (feed it the context pack plus the artifacts you already produced). Re-run `kaddo understand` any time to see this plan again.
